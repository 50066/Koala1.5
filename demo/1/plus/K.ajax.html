3.语法

参数名称 类型 描述
options mix ajax请求配置选项
options可配置项,粗体为默认选项
asynchronous (可选) true/false 同步请求还是异步请求
contentType (可选) 'text/html',"text/javascript","text/xml","application/xml" 请求类型
encoding (可选) UTF-8 请求编码
url (必选) 支持.html,.json,.xml 请求服务器资源地址
readyState (可选) 0 服务器状态码
parameters (可选) 向服务器发送请求url后缀参数
requestHeader (可选) [] 发送请求头信息
method (可选) get set 请求类型
send (可选) null 向服务器发送的数据
loadListener (可选) readyState为1时调用的onreadystatechange监听器
loadedListener (可选) readyState为2时调用的onreadystatechange监听器
ineractiveListener (可选) readyState为3时调用的onreadystatechange监听器
htmlResponseListener(xhr.responseText) (依赖必选) 当请求成功并且响应的Content-Type为text/html时调用的侦听器，这个侦听器将从响应中取得的HTML字符串作为其第一个参数
xmlResponseListener(xhr.responseXML) (依赖必选) 当请求成功并且响应的Content-Type为application/xml或者application/xhtml+html时调用的侦听器，这个侦听器将从响应中取得的XML DOM文档作为其第一个参数
completeListener(arg) (可选) 当上面所有所列的针对Contont-Type为text/html或者text/json时调用的侦听器，这个侦听器总是在成功响应后被调用，可以作为兜底儿的侦听器
errorListener(arg) (可选) 当响应状态值不是200也不是2时调用的侦听器，如果在本地上运行XMLHttpRequest,那么状态值始终是0.
 
返回值 无 无
